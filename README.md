# Data Science Portfolio Projects

This folder contains my data science project notebooks created during the Yandex Practicum training program.  
Each notebook is a standalone educational project focused on a specific topic in the program, such as:

- exploratory data analysis,
- statistical analysis,
- linear models,
- supervised learning,
- machine learning for business,
- numerical methods,
- time series forecasting,
- text machine learning,
- computer vision.

The notebooks include data preparation, modeling, evaluation, and conclusions.

## Projects

- `01_eda_real_estate_listings_spb.ipynb` — EDA of St. Petersburg real estate listings, feature engineering, and pricing factors.
  - Stack: pandas, numpy, matplotlib, seaborn
  - Methods: data quality checks, feature engineering, correlation analysis, analytical reporting
  - Metrics: descriptive stats, correlation coefficients
- `02_stat_analysis_scooter_rental_gofast.ipynb` — statistical analysis of scooter rides and hypothesis testing.
  - Stack: pandas, numpy, scipy, matplotlib
  - Methods: statistical inference, hypothesis testing, confidence intervals
  - Metrics: p-values, confidence intervals
- `03_linear_models_farm_cow_selection.ipynb` — linear/logistic models to select cows based on yield and milk taste.
  - Stack: pandas, scikit-learn, matplotlib
  - Methods: regression & classification pipelines, feature scaling, model evaluation
  - Metrics: MAE, MSE, R2, accuracy, precision, recall
- `04_supervised_learning_customer_activity_prediction.ipynb` — predict customer activity drop and segment clients for actions.
  - Stack: pandas, scikit-learn, SHAP, matplotlib
  - Methods: classification pipelines, model comparison, feature importance, customer segmentation
  - Metrics: F1, ROC-AUC, precision, recall
- `05_ml_for_business_oil_well_location_selection.ipynb` — choose the best region for drilling using ML and risk analysis.
  - Stack: pandas, scikit-learn, numpy, matplotlib
  - Methods: regression, bootstrap, risk modeling, profit simulation
  - Metrics: RMSE, profit/risk distribution, confidence intervals
- `06_numerical_methods_car_price_prediction.ipynb` — predict used car prices with model comparison and tuning.
  - Stack: pandas, scikit-learn, LightGBM, XGBoost, CatBoost
  - Methods: regression pipelines, hyperparameter tuning, cross-validation, model benchmarking
  - Metrics: RMSE, MAE
- `07_time_series_taxi_orders_forecasting.ipynb` — time series forecasting of taxi demand by hour.
  - Stack: pandas, statsmodels, scikit-learn, LightGBM
  - Methods: time series analysis, resampling, feature generation, forecasting
  - Metrics: RMSE
- `08_text_ml_toxic_comments_classification_wikishop.ipynb` — toxic comment classification for moderation.
  - Stack: pandas, scikit-learn, NLTK
  - Methods: text preprocessing, TF-IDF, classification, model evaluation
  - Metrics: F1
- `09_computer_vision_age_estimation_from_photos.ipynb` — age estimation from photos using computer vision.
  - Stack: pandas, TensorFlow/Keras, matplotlib
  - Methods: CNN, image preprocessing, regression
  - Metrics: MAE
